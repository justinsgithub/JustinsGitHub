# basic Linux commands 

> using the command line


## touch
### touch new memory space, aka create new file

|    example    |         description        |
| :------------ | :---------------------------------: |
| touch ~/Documents/new-file.txt | creates new-file.txt in Current users Documents directory |

## mkdir
### make directory

|    example    |         description        |
| :------------ | :---------------------------------: |
|  mkdir ~/NewFolder   | creates NewFolder directory in home directory |
|  mkdir ~/Documents/New-Folder-1/New-Folder-2   | gets denied since New-Folder-1 does not exist yet|
|  mkdir -p ~/Documents/New-Folder-1/New-Folder-2   | "make directory path", creates multiple directories |
|  mkdir /home/my-username/NewDirectory   | creates NewDirectory in home directory |
|  mkdir /NewDirInRootDir   | gets denied because "sudo" command is not being used  |
|  mkdir /home/different-users-username/NewFolder   | gets denied because this is the home directory of a different user|
