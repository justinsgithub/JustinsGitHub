# Get Started With Linux

## Commands to setup a new linux installation

### Getting Started

These commands are for a new (minimal) installation of Ubuntu 21.10, from the terminal.

import Callout from 'nextra-theme-docs/callout'

<Callout>
refresh and update software packages
</Callout>

1. `sudo apt update && sudo apt upgrade`

2. `sudo snap refresh && sudo snap install hello-world && hello-world`

3. `reboot`


<Callout>
install common packages and general setup
</Callout>

1. `ssh-keygen`
in order to retrieve public key for use on github or any other reason:

`cat /home/MYUSERNAME/.ssh/id_rsa.pub`

2. `sudo apt install nginx git gnome-tweaks terminator gnupg`
3. `sudo apt install libncurses-dev`
4. `git config --global user.email "MYEMAIL"` 
5. `git config --global user.name "MYUSERNAME"` 

<Callout>
mongodb from the official repos, enable autostart on system reboots
</Callout>

1. `wget -qO - https://www.mongodb.org/static/pgp/server-5.0.asc | sudo apt-key add -`
2. `sudo touch  /etc/apt/sources.list.d/mongodb-org-5.0.list`
3. `sudo echo "deb [ arch=amd64,arm64 ] https://repo.mongodb.org/apt/ubuntu focal/mongodb-org/5.0 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-5.0.list`
4. `sudo apt-get update`
5. `sudo apt-get install -y mongodb-org`
sudo mkdir -p /var/lib/mongodb
sudo mkdir -p /var/log/mongodb

6. `sudo systemctl start mongod`
7. `sudo systemctl status mongod`
8. `sudo systemctl enable mongod`


<Callout>
python packages
</Callout>

1. `sudo apt install python3-pip python3-selenium python3-pymongo `
