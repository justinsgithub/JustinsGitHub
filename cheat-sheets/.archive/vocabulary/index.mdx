# terms and definitions

## API

> application programming interface
- a collection of code that provides prebuilt functionality
- premade functions that help improve the development process more efficient

## stack data structure

- only allows data to be added or removed from the top
- first in, last out
- last in, first out
- good for something like storing receipts, since the more recent it is, the more chance you may have to refer back to it.
