# Linux commands 

> quick overview

> syntax: command argument(s)

> example: touch my-new-file (touch = command, my-new-file = argument)


|    command    |         description        |
| :------------ | :---------------------------------: |
| touch | creates new file with name of argument|
| mkdir | creates new folder with name of argument|
| echo | prints argument to command line|
| cd |changes to directory specified by argument|
| ls  | lists files and folders in current directory|
| mv  | renames argument 1 to argument 2 or moves argument 1 into directory named argument 2 |
