# NextJS  

> using latest version of node, npm and, Next.JS

## set up Next.JS with Stripe payments

```shell 
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

bash

nvm install 17.0.1

npm install -g npm@8.1.3

npx create-next-app --example with-typescript my-nextjs-app

npm install next@latest 

npm install --save stripe @stripe/stripe-js next


```
