# Express

> web framework for Node.js

> all following examples are performed from the command line using bash

## install

```shell
mkdir my-express-app
```

```shell
cd my-express-app
```

```shell
npm init -y
```

```shell
npm install express
```


## hello-world

```shell
touch index.js
```

```shell
echo '

const express = require("express")
const app = express()
const port = 3000

app.get("/", (req, res) => {
  res.send("Hello World!")
})

app.listen(port, () => {
  console.log(`Example app listening at http://localhost:${port}`)
})

' >> index.js
```

```shell
node index.js
```
